# Angular Housing App

A modern Angular 20 application that simulates a real estate browsing experience with filtering, property details, and mortgage calculation.

## Features

- Property listing with filters (price, location, type)
- Mortgage calculator
- Realtor profile modal
- Responsive UI

## Tech Stack

- Angular 20
- TypeScript
- SCSS
- Docker

## Run Locally

```bash
npm install
ng serve
```
